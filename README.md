# Coding Handbook

This repository is a centralized engineering handbook for language- and stack-specific coding guidance.

## Start Here

- Humans: start with `golang/README.md`
- Agents: start with `AGENTS.md`
- Maintainers routing changes inside the Go handbook: use `golang/maintainer-map.md`

## Current Handbooks

- `golang/` - production-oriented guidance for new Go services, workers, CLIs, and libraries

All detailed guidance lives inside the language-specific handbook directories. The repo root stays intentionally thin so it can route cleanly as more handbooks are added later.
