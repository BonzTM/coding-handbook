# Coding Handbook

This repository is a centralized engineering handbook for language- and stack-specific coding guidance.

## Start Here

- Humans: start with the README of the handbook matching your language — `golang/README.md` or `csharp/README.md`
- Agents: start with `AGENTS.md`
- Maintainers routing changes inside a handbook: use the Change Routing table in that handbook's `AGENTS.md` (`golang/AGENTS.md`, `csharp/AGENTS.md`)

## Current Handbooks

- `golang/` - production-oriented guidance for new Go services, workers, CLIs, and libraries
- `csharp/` - production-oriented guidance for new C#/.NET services, workers, CLIs, and libraries, developed and verified across Windows, Linux, and macOS

All detailed guidance lives inside the language-specific handbook directories. The repo root stays intentionally thin so it can route cleanly as more handbooks are added later.
